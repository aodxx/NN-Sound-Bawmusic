# 🎵 Bawmusic — Band Booking Operating System

ระบบจัดการการจองงานสำหรับวงดนตรีและทีมงานอีเวนต์ ครบวงจรในที่เดียว

ออกแบบให้ใช้งานง่ายเหมือนแอปมือถือระดับพรีเมียม พร้อมธีม **Navy/Gold** สไตล์หรูหรา

---

## ✨ ฟีเจอร์หลัก

| โมดูล | รายละเอียด |
|---|---|
| 📊 **Dashboard** | ภาพรวมงานวันนี้ งานที่จะมาถึง รายได้ ยอดค้างชำระ กิจกรรมล่าสุด |
| 📅 **จัดการการจอง** | ฟอร์มจองงานครบถ้วน พร้อมตรวจสอบคิวและอุปกรณ์ซ้ำซ้อนอัตโนมัติ |
| 👥 **จัดการลูกค้า** | ประวัติการจอง รายได้สะสม ประเภทงานที่ชอบ ค้นหาได้ทันที |
| 📦 **คลังอุปกรณ์** | จัดหมวดหมู่อุปกรณ์ ตรวจสอบจำนวนคงเหลือแบบเรียลไทม์ |
| 📈 **สถิติและวิเคราะห์** | กราฟรายได้รายเดือน อันดับลูกค้า ประเภทงานยอดนิยม |
| 🧾 **สรุปงาน** | สร้างใบสรุปงานสวยงาม แชร์เป็นรูป PNG หรือพิมพ์ได้ทันที |
| ⚙️ **เทมเพลตงาน** | เลือกเทมเพลต (งานแต่ง/บวช/ศพ/คอนเสิร์ต ฯลฯ) เติมอุปกรณ์อัตโนมัติ |

## 🛠 เทคโนโลยีที่ใช้

- **Frontend**: HTML5, Tailwind CSS, Alpine.js, Chart.js, SweetAlert2, Font Awesome
- **Backend**: Google Apps Script (REST API)
- **ฐานข้อมูล**: Google Sheets
- **Hosting**: GitHub Pages
- **PWA**: ติดตั้งลงหน้าจอหลัก ใช้งานแบบออฟไลน์ได้บางส่วน

## 🚀 เริ่มต้นใช้งาน

ดูขั้นตอนติดตั้งแบบละเอียดในไฟล์ **[INSTALL.md](./INSTALL.md)**

สรุปสั้นๆ:
1. คัดลอกโค้ดใน `code.gs` ไปวางใน Google Apps Script ของ Google Sheet ใหม่
2. Deploy เป็น Web App (Who has access: **Anyone**)
3. แก้ `API_URL` ใน `js/api.js` เป็น URL ที่ได้
4. อัปโหลดไฟล์ทั้งหมด (ยกเว้นโฟลเดอร์ `api/`) ขึ้น GitHub Pages
5. เปิดลิงก์บนมือถือ แล้วกด "เพิ่มลงหน้าจอหลัก"

## 📌 หมายเหตุ

- ระบบสร้างชีตฐานข้อมูลและข้อมูลตัวอย่างให้อัตโนมัติในการเรียก API ครั้งแรก
- ปรับแต่งชื่อวง โลโก้ สี และข้อมูลติดต่อได้ที่หน้า **ตั้งค่า** ในแอป
- โค้ด Apps Script ใช้ `var` แทน `const`/`let` เพื่อความเข้ากันได้กับ V8 runtime
